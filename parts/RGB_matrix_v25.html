Description:

Description: This is a large sized RGB LED Matrix with a backpack that has been designed to take SPI serial input and display any graphics you pass it. All the refreshing and communication control is taken care of by the backpack. It's even reprogrammable! 7 software selectable colors are available!

Unit comes with a Red/Green/Blue LED Matrix assembled with a RGB Matrix Backpack Controller.

Check out the LED Coffee Table tutorial!
